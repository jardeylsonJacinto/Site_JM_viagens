# JM website trips
Fictional website of a travel agency.

 ## About Project
This resource was created to put into practice the knowledge acquired in the Udemy Web Development course of a Front-end web project.
  The objective of the project was to create a fictitious website for a travel company using simple technologies such as HTML, in addition to encompassing the concepts of how to structure and position HTML tags.
